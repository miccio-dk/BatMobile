# BatMobile

Sensors-enabled line-following robot. 

Developed as part of a DTU Diplom 4th semester multidisciplinary project.

## Installation

1. Print the PCBs (a breadboard will do too)
2. Cut the chassis pieces
3. Upload the code
4. Assemble everything
5. Run!
